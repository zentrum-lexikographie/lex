# ZDL/Lex – Lexikographic Workbench

## Build Requirements

* [Clojure](https://clojure.org/)
* [Leiningen](https://leiningen.org/)
* [Ansible](https://www.ansible.com/)

## Build and Deploy

    $ make deploy
